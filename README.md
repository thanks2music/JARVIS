# AI With Life

本リポジトリは、「@thanks2music (Yoshiyuki Ito)」のAI活用の進捗や、トライアンドエラーの可視化、実験、検証、また現在の thanks2musicの「AI活用grand design」の記録をまとめる場所とする。

## Why?

何故このリポジトリを作ろうと思ったか、については「爆発的なスピードで進化するLLM/AI業界」に、100%追従する事は混乱である。
しかし、「ClaudeCode」や「MCP」といった”現時点でのAIを使って開発をする際のグランドデザイン”という物が見えてきており、
この記録をGitHubなどクラウドに残しておきたい、と思ったためです。クラウドに残すのは、可視化と環境が変わった際にも再構築がしやすくするためである。
